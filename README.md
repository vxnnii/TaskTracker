# Student Task Tracker

## A simple console-based task manager built in Java. Lets you add, view, complete, and remove tasks right from the terminal.

### Features


- Add tasks – quickly jot down anything you need to get done
- View tasks – see everything in your list with its current status
- Mark complete – flag a task as done without deleting it
- Remove tasks – clear out tasks you no longer need
- Input validation – handles bad input (like letters instead of numbers) without crashing
